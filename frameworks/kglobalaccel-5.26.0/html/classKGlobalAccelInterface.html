◆ grabKey()

virtual bool KGlobalAccelInterface::grabKey ( int  key,
bool  grab 
)
pure virtual

This function registers or unregisters a certain key for global capture, depending on grab.

Before destruction, every grabbed key will be released, so this object does not need to do any tracking.

Parameters
keythe Qt keycode to grab or release.
grabtrue to grab they key, false to release the key.
Returns
true if successful, otherwise false.

Implemented in KGlobalAccelImpl, KGlobalAccelImpl, and KGlobalAccelImpl.
