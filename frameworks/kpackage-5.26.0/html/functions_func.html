About

Library to load and install packages of non binary files as they were a plugin

Maintainer
Marco Martin
Supported platforms
Linux, MacOSX, Windows
Community
IRC: #kde-devel on Freenode
Mailing list: kde-frameworks-devel
Use with CMake
find_package(KF5Package)
target_link_libraries(yourapp KF5::Package)
Clone
git clone git://anongit.kde.org/kpackage-5.26.0.git
Browse source
