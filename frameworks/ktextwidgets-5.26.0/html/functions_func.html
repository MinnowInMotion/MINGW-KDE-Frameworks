About

Advanced text editing widgets

Maintainer
Laurent Montel
Supported platforms
Linux, MacOSX, Windows
Community
IRC: #kde-devel on Freenode
Mailing list: kde-frameworks-devel

target_link_libraries(yourapp KF5::TextWidgets)
Use with QMake
