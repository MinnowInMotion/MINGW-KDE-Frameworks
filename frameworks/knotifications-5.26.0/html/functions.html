About

Abstraction for system notifications

Maintainer
Martin Klapetek
Supported platforms
Linux, MacOSX, Windows
Community
IRC: #kde-devel on Freenode
Mailing list: kde-frameworks-devel
Use with CMake
find_package(KF5Notifications)
target_link_libraries(yourapp KF5::Notifications)
Use with QMake
QT += KNotifications
