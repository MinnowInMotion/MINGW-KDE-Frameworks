Widgets for configuration dialogs

Maintainer
The KDE Community
Supported platforms
Linux, MacOSX, Windows
Community
IRC: #kde-devel on Freenode
Mailing list: kde-frameworks-devel

target_link_libraries(yourapp KF5::ConfigWidgets)
