About

Provides integration of QML and KDE Frameworks

Maintainer
Marco Martin
Supported platforms
Linux, MacOSX, Windows
Community
IRC: #kde-devel on Freenode
Mailing list: kde-frameworks-devel
Use with CMake
find_package(KF5Declarative)
target_link_libraries(yourapp KF5::Declarative)
Use with QMake
QT += KDeclarative
