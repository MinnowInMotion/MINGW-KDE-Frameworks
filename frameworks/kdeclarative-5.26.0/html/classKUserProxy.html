Detailed Description

KUserProxy (exposed as KUser to the QML runtime) is an object allowing read-only access to the user's name, os and version and the configured user image.

This object can be used to personalize user interfaces.

Example usage:

import org.kde.kcoreaddons 1.0 as KCoreAddons
[...]
Item {
[...]
KCoreAddons.KUser {
id: kuser
}
Image {
id: faceIcon
source: kuser.faceIconUrl
[...]
}
Text {
text: kuser.fullName
[...]
}
}

KUser provides read-only access to the user's personal information
